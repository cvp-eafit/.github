# Valor Público · Universidad EAFIT

> Investigamos, creamos y acompañamos proyectos y procesos estratégicos para mejorar problemas públicos: conocimiento para el país posible.

Somos el [Centro de Valor Público](https://www.eafit.edu.co/centros-estudio-incidencia/valor-publico) de la Universidad EAFIT. Desarrollamos investigación aplicada y consultoría para el fortalecimiento del sector público en Colombia.
